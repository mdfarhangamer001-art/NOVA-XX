# 🧰 Available Tools

A complete reference of all tools IRIS can execute autonomously.

## System & File Management

- **Open/Close App:** Process control.
- **Read/Write/Manage File:** Local disk access.

## Vision & Automation

- **Phantom Typer:** Clipboard injection.
- **Click/Scroll Screen:** Nut.js powered exact targeting.
- **Screen Peeler:** Tesseract.js OCR extraction.

## Web & Deep Research

- **Deep Research:** Tavily-powered web crawling.
- **Hack Live Website:** DOM manipulation.
