# 📖 Glossary

- **Neural OS:** The conceptual framework of IRIS as an OS-level agent layer.
- **IPC Bridge:** Inter-Process Communication between React and Node.js.
- **ASAR:** Atom Shell Archive. The packaged format for Electron apps.
- **V8 Bytecode:** The compiled, unreadable format of the protected backend code.
- **RAG Oracle:** The local vector-database reasoning engine powered by LanceDB.
